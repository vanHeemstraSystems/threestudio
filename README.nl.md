**Samenvatting**

Threestudio is een uniform raamwerk voor het maken van 3D-inhoud op basis van tekstprompts, afzonderlijke afbeeldingen en afbeeldingen met een paar opnames, door 2D-modellen voor het genereren van tekst naar afbeeldingen te gebruiken.

Gegenereerd met[Ritr](https://app.rytr.me)
